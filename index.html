VRC

Virtual Republic of code

The Virtual Republic of Code is a democratic Republic for developers based in Munich, Germany.

The current President of the Republic is Ixodev. The current Prime Minister of the Republic is Kubik.

A quick presentation of the VRC This Republic is a place where developers can vote, choose laws, code, discuss, and ask questions about specific programming topics. There are three official functions in the VRC : GOV (Government) President of the Republic : Ixodev Prime Minister : Kubik Ministers : Ixodev, Kubik. FCT (Public functions) Policemen Soldiers Coding Teachers Inspectors CIT (Citizens) Normal developers There are three official speaking languages in the Republic : English German French There are four official programming languages in the Republic : C/C++ Java JavaScript There are four ministers : Ixodev President of the Republic Army Leader C/C++/Java/Python minister Theoritical Mathematics, Physics and Biology Minister Culture Minister You can see the Constitution on the other Constitution repositories.
